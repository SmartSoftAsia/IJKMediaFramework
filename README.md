# IJKMediaFramework



## Installation

IJKMediaFramework is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

``` ruby
platform :ios, '8.0'
use_frameworks!

pod "IJKMediaFramework"
```
